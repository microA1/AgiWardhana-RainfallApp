Rainfall Monitoring Menu

This menu displays the latest rainfall data from ARG, AWS, and AAWS sites, transmitted to the server in real time. Rainfall data is automatically categorized based on meteorological rainfall levels, with each category distinguished by different colors for easier identification.

Rainfall Distribution Map Menu

This menu presents rainfall data in a map format, with points representing site locations. The data is color-coded to indicate the current rainfall intensity. Additionally, this menu provides an option to overlay the map with Himawari-9 IR data for enhanced visualization.

Notification Alert Feature

Notifications will be triggered when rainfall exceeds 50.0 mm, 100.0 mm, and 150.0 mm. All issued notifications will be recorded and displayed within the application.
